## [1.0.0] - 2026-02-15

### Added

- Bookend skill with full protocol: morning briefing, auto-checkpoints, evening wrap, nightly builds
- Plugin structure with `.claude-plugin/plugin.json` for marketplace compatibility
- `/setup` slash command for automated workspace initialization
- YAML frontmatter on SKILL.md with auto-trigger phrases
- Template files: `state-current.md`, `memory-daily.md`, `MEMORY.md`, `AGENTS-snippet.md`
- Real-world examples: session start, session end, populated state file
- Reference documentation: patterns and anti-patterns
- Community files: CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md
- GitHub templates: bug report, feature request, pull request
- CI workflow: markdown linting and link checking
- Ko-fi funding configuration
