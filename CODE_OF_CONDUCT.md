# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In summary: be respectful, be constructive, and be welcoming to all contributors regardless of background or experience level.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it via [GitHub Security Advisories](https://github.com/rockywuest/bookend-skill/security/advisories/new) or by contacting the maintainer directly.

## Enforcement

Project maintainers are responsible for clarifying standards and taking appropriate action in response to any behavior they deem inappropriate, threatening, or harmful.
